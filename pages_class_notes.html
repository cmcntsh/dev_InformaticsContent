VSCode

  • Install VSCode: https://code.visualstudio.com/download
  • Visual Studio Code Intro & Setup Video (39 minutes): https://www.youtube.com/watch?v=fnPhJHN0jTE
    • Download and install VSCode before watching the video.
    • He talks about configuration at the beginning. It's OK to just watch. Try what you want, though.
    • The narrator may mention several things you've never heard of. Don't worry too much about things other than seeing the parts of VSCode and how to use it.
    • At 23:20 he mentions Live Server. Go ahead and install that.
    • At 31:50 he mentions Git and version control. We'll use that again later.
